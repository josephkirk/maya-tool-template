# maya-tool-template
Template for a basic Qt tool in Maya

![tool header image](docs/header_image.png)

Requires Qt.py


# Install options

<pre>
Run install_maya_mod.bat (will create a .mod file in your maya/modules folder)
Restart Maya
</pre>

# Start the tool
<pre>

from TOOL_NAME import TOOL_NAME_UI
TOOL_NAME_UI.main()

</pre>
